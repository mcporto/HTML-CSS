Principais Formatações

Negrito / Destaque

Nessa frase termos um termo em negrito, usando a tag (b)(não semantica).

Nessa frase temos um termo em destaque usando a tag (strong)(semantico).

Italíco / Enfase

Podemos usar a tag (i) para deixar em italico, porem sem semantica.

Usamos a tag (em) para deixar em italico tbm, porem com semantica.

Texto Marcado

Podemos usar a tag (mark) para marcar o texto.

Podemos mudar a cor marcada desse texto no css com a tag (style) e (background) selecionando a cor.

Texto Grande e Pequeno

Nessa frase temos texto grande e temos texto pequeno.

Texto Deletado e Inserido

Podemos desconsiderar um texto usando a tag (del), como nesse exemplo

Podemos tbm inserir um sublinhado usando tag (ins) com semantica, e usando a tag (u) sem semantica.

Texto Sobrescrito e Subscrito

Para sobrescrever um texto usamos a tag (sup) como no exemplo de: x20+3

Também podemos utilizar a tag (sub) para subscrever um texto, como no exemplo : H2O
